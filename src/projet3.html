/Streaming website

Presentation

This project involved creating a web application similar to Netflix or Prime Video to view movie trailers, create user profiles, add favorite movies, etc. The goal was to create two separate components: a "showcase site" for users and a "back office" for administrators. I used tools such as PHP/MySQL to create a functional structure between the front-end and back-end of the application for technical realization.

The second step of the project was to modify the graphic style of my streaming site by taking inspiration from an existing platform. By carefully examining visual features, fonts, color palettes, and element layout
